# Mathematical Formulation

## Functional Input and Irregular Sampling
Each sample is a multivariate profile \(X_i=\{(t_{ik}, x_{ik})\}_{k=1}^{K_i}\), with mask \(m_{ik}\), interval \(\Delta t_{ik}\), and normalized time \(\tau_{ik}\).

## SNN-ODE Cell
SNN-ODEATT maintains membrane and synaptic states. A generic continuous form is:
\[
\dot s(t)=f_s(s(t), x(t), \Delta t),\quad
\dot m(t)=f_m(m(t), s(t), x(t), \Delta t).
\]
Spikes are generated by a smooth surrogate \(s_t=\sigma_\beta(m_t-\vartheta)\), followed by soft reset \(m_t^*\).

## Attention
Causal attention only attends to previous/equal time points. Time-aware bias or weights encode irregular intervals.

## Loss
\[
\mathcal L=\omega_r\mathcal L_{rec}+\omega_p\mathcal L_{pred}+\omega_s\mathcal L_{stability}.
\]

## Scoring
Mahalanobis, Sinkhorn divergence, Gaussian W2, and sliced Wasserstein scoring are implemented under `src/snnodeatt/scoring`.
