# Sinkhorn Notes

Post-hoc Sinkhorn divergence compares hidden trajectory empirical measures against a normal reference measure. In wafer exp4, post-hoc global Sinkhorn was weaker than Mahalanobis/rich scoring, likely because the hidden geometry was trained for latent covariance separation rather than OT geometry. Sinkhorn remains useful for future external datasets with state accumulation or trajectory-shape anomalies.
